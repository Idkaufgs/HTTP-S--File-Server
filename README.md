The best self made Fileserver ever made (/jk).

If you somehow randomly stumble onto this small repository, hi I'm a self taught dev.
This is a small project of (and originally for) myself.
Feel free to use and modify this as you wish. And feel free to open issues for all your needs (whether it's a feature request or a bug I'll eventually come around to looking at it).

It's currently only the logic code, so using it requires some manual setup from your side ( I am planning on eventually adding an installer).

Note that it currently only has support for Windows and currently only gets tested on the Windows Appache Server, so if you want to use something else note that you might have to tweak things or that it might not work at all.
